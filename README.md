## 🎀 Features

- 🍓 **Custom Cursor** — Animated strawberry cursor with emoji trail effect
- 🌸 **Floating Particles** — Background floating emoji particles (🌸 💕 ✨ 🎀 ⭐)
- ✨ **Scroll Animations** — Cards fade in smoothly on scroll using IntersectionObserver
- 💖 **Interactive Skill Tags** — Click/hover effects with sparkle burst
- 🎵 **Music Note Effect** — Click the music trigger to spawn floating musical notes
- 🖼️ **Certificate Gallery** — Hover-reveal overlay on certificate images
- 💗 **Profile Image Click** — Heart burst animation on profile photo click
- 🎮 **Easter Egg** — Konami code (↑↑↓↓←→←→BA) activates a rainbow + emoji shower
- 📱 **Responsive Design** — Mobile-friendly layout

## 🚀 Getting Started

1. **Clone or download** this repository
2. Make sure all image files are in the **same folder** as `index.html`
3. Open `index.html` in any modern browser
4. No build tools or dependencies needed — it's 100% vanilla!

```bash
# Example with VS Code Live Server
code .
# Then right-click index.html → Open with Live Server
```
